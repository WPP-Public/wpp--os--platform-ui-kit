# Components Library

This is a monorepo for UI web-components built with [Stencil.js](https://stenciljs.com/) and [Storybook](https://storybook.js.org/), where you can see those components in action.

For more information about how you can set up the design system please refer to our official documentation [here](https://developers.os.wpp.com/docs/design-system/about-ods/developers/).
