## 🚀 Getting Started

Before the first run, you will need to install dependencies:

```bash
npm i
```

> **Note!** No need to install dependencies in nested directories.

To build component library with React and Angular wrappers use
- `npm run build`

If you want to run it in watch mode use
- `npm run build:watch`

After that use one of these commands for local development depends on your goals:
  - `npm run start:angular-example`
  - `npm run start:react-example`
  - `npm run start:vue-example`
  - `npm run start:storybook`
