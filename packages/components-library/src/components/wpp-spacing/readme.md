## Spacing

Spacing controls the amount of space for details within a component or detailed UI.
Spacing helps organize content, providing the consistent visual balance that makes the user interface easier to scan.

# Usage
In order to add spacing, one should add classes to the particular elements.
For example:

```html
<div class="wpp-spacing-2-top" />

<div class="wpp-spacing-2-right" />

<div class="wpp-spacing-2-bottom" />

<div class="wpp-spacing-2-left" />
```
